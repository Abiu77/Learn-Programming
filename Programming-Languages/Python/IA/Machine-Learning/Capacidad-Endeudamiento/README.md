## Ejemplo:

+ Capacidad de endeudamiento.
  
  + Ingresos mes = 2000

  + Gastos mes = 1000

  + `Tatal * 35% = Capacidad de endeudamiento`.

  + `1000 * (0.35) = 350`

| 3000 | 1000 | 1 |
